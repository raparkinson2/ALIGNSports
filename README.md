# Hockey Team Manager

A mobile app for recreational hockey teams to manage schedules, check-ins, lineups, and team photos.

## Features

### Schedule Tab
- View upcoming games with opponent, date, time, and rink location
- See jersey color for each game
- Quick view of check-in status and who's bringing the beer bag
- Tap any game to view full details

### Game Details
- Full game information including time, location, and jersey color
- Tap location to open in Maps for directions
- Check in/out for games
- View tonight's lineup organized by position (Forwards, Defense, Goalie)
- See who's assigned beer bag duty

### Roster Tab
- View all team players organized by position
- See each player's jersey numbers and colors they own
- Add new players with name, number, position
- Select which jersey colors each player owns (for game planning)

### Photos Tab
- Team photo gallery
- Add photos from camera roll or take new photos
- Share game memories with the team

## Tech Stack
- Expo SDK 53 / React Native
- Expo Router for navigation
- Zustand for state management
- NativeWind (Tailwind CSS) for styling
- React Native Reanimated for animations
- Lucide icons

## Design
- Dark theme with ice blue (#67e8f9) accents
- Hockey rink-inspired color palette
- Smooth animations and haptic feedback
- Mobile-first, thumb-friendly design
